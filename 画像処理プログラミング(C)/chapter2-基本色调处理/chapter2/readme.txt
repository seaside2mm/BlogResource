　このプログラムをコンパイルするには、以下のように指定します（gccの場合）。
Chapter1ディレクトリの「bmp.c」「image.c」「image.h」を同じディレクトリに
置きます。

　　gcc -o [実行ファイル名] [ソースファイル名] bmp.c image.c

　なお、list2_06.cについては、

　　gcc -o [実行ファイル名] [ソースファイル名] bmp.c image.c solve.c

となります。

　ソースコード内で「math.h」をインクルードしている場合、gccでは「-lm」オプ
ションが必要になります。
